Teste de Clickjacking

Análise de vulnerabilidade para https://mudream.online

O que é Clickjacking?

Clickjacking é uma técnica maliciosa que engana os usuários para que cliquem em algo diferente do que percebem, potencialmente revelando informações confidenciais ou tomando controle de seus computadores.

Este teste verifica se um site permite ser incorporado em um iframe, tornando-o potencialmente vulnerável a ataques de clickjacking.

Teste 1: Iframe Simples

Tentativa de carregar o site em um iframe. Se o site carregar, pode ser vulnerável.

Teste 2: Iframe com Overlay

Tentativa de clicar em um botão sobreposto ao iframe. Se for possível interagir com o site através do overlay, há vulnerabilidade.

Teste 3: Iframe Oculto

Tentativa de carregar o site em um iframe invisível. Se o site carregar, pode ser vulnerável.

Resultado do Teste: VULNERÁVEL

Como se proteger contra Clickjacking?

  • Implementar o cabeçalho X-Frame-Options
  • Utilizar Content Security Policy (CSP) com frame-ancestors
  • Implementar JavaScript Frame Busting (não tão confiável)
  • Validar sempre as origens de requisições
